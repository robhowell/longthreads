# Spacing

A spacing scale is used across the app that provides a fixed set of spacing options for padding, margins and gaps. By limiting ourseves to a set of fixed values rather than using any specific number of pixels, we ensure consistency across components, speed up design and make it easier to maintain the app.

export const Space = ({ sizeLabel, value }) => {
  const valueNum = parseInt(value.replace('px', ''), 10);
  return (
    <div style={{ display: 'flex' }}>
      <div style={{
        fontSize: '10px',
        minWidth: '145px',
        width: '145px'
      }}><strong>Size {sizeLabel}</strong>: {value} (16px * {valueNum / 16})</div>
      <div style={{
        backgroundColor: 'darkblue',
        height: '12px',
        marginBottom: '5px',
        width: value
      }}></div>
    </div>
  );
};
