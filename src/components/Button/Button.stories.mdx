#  Button

Note: While `<button>` elements in HTML have a default `type` value of `submit`, this is less frequently used in modern single page applications, so instead the Button component `type` prop has a default value of `button`.

If you want to render an element that navigates to a page that is styled as a `Button`, pass in `a` as the `as` prop, then you can use href to navigate as a standard link.
